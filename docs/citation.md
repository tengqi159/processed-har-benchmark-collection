# Citation

There is not yet a dedicated paper for this processed data release. For now, cite the data repository itself, the original dataset papers for the subsets you use, and relevant HAR method papers from the maintainer's publication line.

Recommended citation order:

1. Cite the processed data release below.
2. Cite each original benchmark dataset used in your experiments.
3. Cite the related Qi Teng HAR papers below when they are relevant to your method or comparison context.

## Data Release
