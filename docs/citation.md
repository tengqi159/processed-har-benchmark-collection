# Citation

There is no standalone paper or DOI for this processed dataset collection yet. Do not cite an unpublished data-release entry.

If you use these processed files, cite the original public dataset paper for each subset you use, and cite the relevant Qi Teng HAR papers below. You can mention the GitHub or Hugging Face URL in a data-availability statement, but the scholarly citations should be real papers.
