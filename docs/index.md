# ReadyHAR: Ready-to-Use HAR Datasets

ReadyHAR provides preprocessed Human Activity Recognition datasets as model-ready NumPy train/test arrays.

## Download

- Hugging Face: `https://huggingface.co/datasets/shenjianmozhu/preprocessed-har-datasets`
- Google Drive: `https://drive.google.com/drive/folders/1Qe8AQ8S2V4_uBIvC3Pv4WRmv-EPWERjG?usp=drive_link`
- GitHub: `https://github.com/tengqi159/ready-to-use-har-datasets`

## Current Folders

`uci`, `unimib`, `pamap2`, `wisdm`, `oppo`, and `WSBHA` are currently available under `datasets/`.

## Citation

There is no standalone dataset paper yet. Cite the original dataset papers and the real Qi Teng HAR papers listed in `docs/citation.md`.
